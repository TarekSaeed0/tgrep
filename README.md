## Features

- Searching in a single file or in a directory recursively.

## Building

```sh
git clone https://github.com/TarekSaeed0/tgrep.git
cd tgrep
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```
